# Data Cleaning & Reporting Automation

## Project Overview

This project automates data cleaning and reporting workflows using Python and Pandas.

## Features

* Missing value handling
* Duplicate removal
* Data preprocessing
* Automated summary report generation
* Visual reporting using charts

## Tools Used

* Python
* Pandas
* Matplotlib
* Google Colab

## Dataset

Employee dataset containing missing values, duplicates, and inconsistencies.

## Outcome

Successfully automated data cleaning, generated reports, and created visual summaries to improve reporting efficiency.
